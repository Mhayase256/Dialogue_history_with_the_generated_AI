# Dialogue_history_with_the_generated_AI
生成AIとのやり取りの履歴を掲載しています

# このリポジトリの目的  
 複数利用している生成AIとのやり取りを一括管理するために作成しました。  
 生成AIとのやり取りのうち、生成AI側からドキュメント形式で回答があり、かつ、長文の回答を掲載しています。  
+ 掲載している内容
	+ やり取りした内容
	+ やり取りした日付
 	+ やり取りした生成AI
	+ やり取りの内容
		+ こちらから問い合わせた内容は”Q"を、生成AIからの回答を"A"と表記しています

## 注意点
日時は日本標準時(JST)基準となります。協定世界時との時差は9時間(UTC+9)となります。
